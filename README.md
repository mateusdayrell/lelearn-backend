# Llrn-backend

### Clone the repository
```
git clone https://github.com/mateusdayrell/llrn-backend.git
```

### Access the project folder
```
cd llrn-backend
```


### Install the dependencies
```
npm install
```

### Run the migrations
```
npx sequelize db:migrate
```

### Run the seeders
```
npx sequelize db:seed:all
```

### Serve the aplication
```
npm run dev
```
